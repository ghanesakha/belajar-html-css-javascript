HTML (HyperText Markup Language)

HTML adalah fondasi dari setiap halaman web yang digunakan untuk membuat struktur dan konten halaman

  1. Struktur Dasar
    Setiap halaman HTML memiliki struktur dasar dengan elemen <head>, <title>, <body>, dan <h1>.
  2. Hyperlink
    Membuat tautan ke halaman lain menggunakan tag <a>.
  3. Gambar
    Menyisipkan gambar dengan tag <img> dan mengatur atribut seperti src dan alt.

CSS (Cascading Style Sheets)

CSS digunakan untuk mengatur tampilan dan gaya halaman web merupakan bagian penting yang menghasilkan desain yang menarik.

  1. Selektor
    Menggunakan selektor untuk memilih elemen yang ingin distyling.
  2. Properti dan Nilai
    Properti CSS, seperti color dan font-size, memiliki nilai yang mengendalikan aspek visual elemen.
  3. Tata Letak
    Menggunakan properti seperti display, position, dan float untuk mengontrol tata letak elemen.
  4. Kelas dan ID
    Memberikan kelas atau ID ke elemen untuk menerapkan gaya khusus.
  5. Box Model
    Memahami konsep kotak model yang terdiri dari margin, border, padding, dan content.

JavaScript

JavaScript adalah bahasa pemrograman yang digunakan untuk membuat halaman web interaktif.

  1. Variabel
    Menyimpan data dalam variabel menggunakan kata kunci var, let, atau const.
  2. Fungsi
    Membuat fungsi untuk mengelompokkan kode yang dapat dipanggil kembali.
  3. Event Handling
    Menangani interaksi pengguna seperti klik tombol dengan mendengarkan event.
  4. Manipulasi DOM
    Menggunakan JavaScript untuk mengubah elemen dan konten di halaman dengan mengakses Dokumen Objek Model (DOM).
  5. Percabangan dan Perulangan
    Menggunakan kondisi if untuk percabangan dan perulangan for atau while untuk mengulangi tindakan tertentu.
